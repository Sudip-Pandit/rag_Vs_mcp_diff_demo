MCP Execution Flow

graph TD A[User Query] --> B[LLM parses intent
and decides which tool to call] B --> C[MCP routes the call
to the registered tool/API] C --> D[Live system returns
real-time data] D --> E[LLM synthesizes result
into final answer]

📌 When to Use MCP

Scenario Use MCP?
Check live pipeline run status ✅ Yes
Fetch current row count from a Delta table ✅ Yes
Trigger an alert or restart a job ✅ Yes
Answer "What is SCD Type 2?" ❌ No — that's a knowledge question
Summarize our data contract policy ❌ No — use RAG
